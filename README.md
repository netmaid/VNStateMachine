## Requirements

- Objective-C
- iOS 6.0 or later
- Mac OS 10.9 or later

## Installation

#### [CocoaPods](https://github.com/cocoapods/cocoapods)

- Insert `pod "VNStateMachine"` to your Podfile.
- Run `pod install`.
